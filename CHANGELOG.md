# Changelog

## v1.1.0 (2026-05-26)

### 新增

#### 1. 文案表达力升级
- 3秒钩子规则升级：增加"处境锚定版"钩子类型
- 新增钩子自检清单（4项检查：具体对象/真实处境/画面动作/去空泛）
- 新增"去AI味"检查规则（删除模板感/热词堆砌/泛泛价值判断）

#### 2. 角色深度分析
- 输出新增"角色三维分析"字段
- 角色弧光：起点→转折点→终点（状态/动机/代价）
- 关系张力：表面冲突→深层冲突→不可调和性

#### 3. 视频化输出
- 输出新增"视频化建议"字段
- 解说稿：基于高能版生成的口播脚本
- 分镜表：镜头编号/景别/画面/情绪/时长/运镜
- BGM建议 + 转场建议
- 景别说明：WS/MS/CU/ECU

### 修改
- 能力边界更新：新增"角色三维分析""视频化输出"能力声明
- 输出YAML结构扩展：新增角色三维分析、视频化建议字段
