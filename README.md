# awkn-爽点压缩器 / High-Point Compressor

> Compress novels, short dramas, and comics into a 1-minute highlight reel
> 把小说、短剧、漫画压缩成 1 分钟爽点速览

---

## 简介 / Overview

### 中文

把一部 80 集的长篇内容，压缩成观众 1 分钟就能看完所有爆爽点的版本。

适用于：长篇小说/漫画压缩、短剧解说稿生成、二创素材提取。

### English

Compress an 80-episode long-form piece into a 1-minute version that lets viewers experience every highlight moment without the filler.

Perfect for: long novel / comic compression, short-drama narration scripts, derivative material extraction.

---

## 核心能力 / Core Capabilities

| 能力 / Capability | 说明 / Description |
|-------------------|---------------------|
| 3秒钩子 / 3-sec Hook | 一句话爆爽点 / One-sentence highlight |
| 30秒剧情入口 / 30-sec Intro | 快速了解核心爽线 / Quick grasp of the core highlight line |
| 1分钟高能版 / 1-min Highlight | 4个一级爽点串联 / 4 Tier-1 points chained |
| 3分钟高能版 / 3-min Highlight | 10-15个核心爽点 / 10-15 core points with rhythm |
| 结局速通 / Ending Rush | 3个核心反转 / 3 core reversals |
| 角色关系图 / Relationship Map | 谁和谁有仇 / Who has beef with whom |
| 角色三维分析 / 3D Character Analysis | 弧光+关系张力 / Arc + Relationship tension |
| 视频化建议 / Video Adaptation | 解说稿+分镜表+BGM+转场 / Script + Shot list + BGM + Transitions |
