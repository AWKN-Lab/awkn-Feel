# awkn-爽点压缩器

把小说、短剧、漫画压缩成"高爽点消费版"。

## 文件结构

```
awkn-爽点压缩器/
├── SKILL.md                           # 技能主文件
├── CHANGELOG.md                       # 版本更新记录
├── references/
│   └── 01_high_point_categories.md    # 爽点分类详细说明
└── samples/
    └── 01_sample_novel_compression.md  # 小说爽点压缩示例
```

## 功能

- 3秒钩子生成
- 30秒剧情入口
- 1分钟高能版 / 3分钟高能版
- 结局速通版
- 角色关系图
- 角色三维分析
- 高能片段索引
- 视频化建议
